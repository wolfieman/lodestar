# assets/

Visual media for the project README — produced by the maintainer (see
[`../docs/media-guide.md`](../docs/media-guide.md) for exact steps), then referenced from
the top-level `README.md`.

Suggested files (optional — the README renders fully without them, using Mermaid diagrams):

| File | What it is | How to make it |
|---|---|---|
| `lodestar-overview.png` | An infographic of the architecture / value prop | NotebookLM, Gamma, or Canva — see media-guide |
| `lodestar-overview.mp4` | A ~3-min video overview (à la the EV-Pulse / PWC projects) | NotebookLM **Video Overview** — see media-guide |
| `lodestar-demo.gif` | A terminal recording of `uv run lodestar` answering a question | `vhs` or asciinema — see media-guide |

> Large media (`*.mp4`) should be tracked with Git LFS if committed (`git lfs track "*.mp4"`),
> or linked externally to keep the repo lightweight.
